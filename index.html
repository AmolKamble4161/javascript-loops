JavaScript Loops

Loops are used to execute a piece of code again and again. Loops in JavaScript allow us to repeat a block of code multiple times until a certain condition is met. They help avoid writing the same code again and again.

  1. For Loop
  2. While Loop
  3. Do-While Loop
  4. For-of Loop
  5. For-in Loop

1] For Loop

A for loop in JavaScript is used to repeat a block of code a certain number of times.

Syntax:

for(initialization; condition; increment/decrement) {
    // code to execute
}
  • initialization â†’ set a starting value (e.g., let i = 0)
  • condition â†’ loop runs while this is true (e.g., i < 5)
  • increment/decrement â†’ updates the counter each time (e.g., i++)

Example: Print sum of first 10 numbers

let sum = 0;
for (let i = 1; i <= 10; i++) { // Here we give nth no as 10
    sum += i;
}
console.log(sum);

2] While Loop

A while loop is used to repeat a block of code as long as a condition is true.

Syntax:

    // code to execute
}
  • condition â†’ checked before each loop run.
  • If itâ€™s true, code runs.
  • If itâ€™s false, loop stops.

Example: Print sum of first 20 numbers

let b = 1;
let add = 0;
while (b <= 20) {
    add += b;
    b++;
}
console.log(add);

4] For Of Loop

The for...of loop is used to iterate over iterable objects like arrays, strings, maps, sets, etc. ðŸ‘‰ It gives you the value of each element (not the index).

Syntax:

for (let element of iterable) {
    // code to execute
}
  • Works only on iterables (Array, String, Map, Set, etc.).
  • Gives values directly (not indexes).
  • If you need indexes, use for...in or array.entries() with for...of.

Example:

The for...of loop is mainly used on Array or String to execute all the values of elements or characters. 

let str = "HelloWorld"; // declare a string
for (let i of str) {
    console.log("i =", i);
}

Output: Prints all characters of the string one by one

5] For In Loop

The for...in loop is used to iterate over the keys (properties or indexes) of an object or array. ðŸ‘‰ It gives you the key/index, not the value directly.

Syntax:

for (let key in object) {
    // code to execute
}
  • Use for...in when you want keys/indexes.
  • Use for...of when you want values.

Example:

let person = { name: "Amol", age: 25, city: "Pune" };
for (let key in person) {
    console.log(key, ":", person[key]);
}

Output:

name : Amol
age : 25
city : Pune
